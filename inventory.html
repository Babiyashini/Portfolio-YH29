Project Overview

This project presents a complete, cloud-native solution to modernize pharmacy operations. It is a web-based Pharmacy Inventory Management System designed to tackle critical challenges like medicine stockouts, expiry losses, and inefficient manual processes for small and medium-sized pharmacies.

Key Features & Capabilities

Core Management

  • Efficiently manage medicine records, track sales, generate invoices, and manage supplier information.

Predictive Analytics

  • Integrated machine learning module (using Linear Regression) forecasts future medicine demand based on historical sales data, aiding in smarter purchasing decisions and reducing waste.

Modern DevOps & Cloud Architecture

  • Automated Deployment (CI/CD): A pipeline using GitHub Actions automatically deploys code changes to the cloud in under 30 seconds.
  • Containerization: The application is packaged with Docker for consistent performance across all environments.
  • Cloud Infrastructure: Deployed on AWS (EC2, RDS, S3) using Terraform (Infrastructure as Code) for a reproducible and scalable setup.
  • Self-Healing & Security: An auto-healing script ensures the application restarts automatically if it crashes, and HTTPS security is implemented with an SSL certificate from Let's Encrypt.
